# İş Analisti Quiz Uygulaması

Bu proje, iş analisti mülakat hazırlığı için hazırlanmış tek sayfa bir quiz uygulamasıdır. Uygulama HTML, CSS ve JavaScript ile hazırlanmıştır ve herhangi bir ek sunucuya ihtiyaç duymadan doğrudan tarayıcıda çalışır.

## Özellikler

- Başlangıç ekranı ve quiz ekranı arasında geçiş
- Sol tarafta modern, açılır/kapanır sidebar
- Soruların kategori, zorluk ve tip bilgileriyle gösterimi
- Süre takibi ve zaman dolunca otomatik bitirme
- Sonuç ekranında başarı oranı, kategori bazlı istatistikler ve öneriler
- Sonuç ekranından ana ekrana dönme
- Responsive tasarım

## Kullanım

1. `index.html` dosyasını tarayıcıda açın veya uygulamayı GitHub Pages üzerinden ziyaret edin:
   - https://salihakyuz.github.io/quizApp/
2. Başlangıç ekranında quiz modunu seçin.
3. `☰ Sorular` butonuna tıklayarak sidebar'ı açıp kapatın.
4. Soruları cevaplayın ve sonuç ekranında `Ana ekrana dön` butonuna basın.
